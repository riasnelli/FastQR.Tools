# FastQR.Tools - WiFi QR Code Generator

A free, secure, and client-side WiFi QR code generator. Create WiFi password QR codes instantly for WPA, WEP, and open networks.

## Features

- ✅ Generate WiFi QR codes instantly
- ✅ Support for WPA, WEP, and open networks
- ✅ Client-side encryption - no data stored
- ✅ Download as PNG or SVG
- ✅ Works offline
- ✅ Modern Material Design UI
- ✅ Responsive design
- ✅ SEO optimized

## How to Use

1. Enter your WiFi network name (SSID)
2. Enter your WiFi password
3. Select security type (WPA/WPA2, WEP, or Open)
4. Toggle "Hidden Network" if applicable
5. Click "Generate QR Code"
6. Download as PNG or SVG

## Technology Stack

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- QRCode.js library

## Installation

Simply clone or download this repository and open `index.html` in a web browser. No build process or dependencies required.

```bash
git clone <repository-url>
cd qr
open index.html
```

## Browser Support

Works on all modern browsers that support:
- ES6 JavaScript
- CSS Grid and Flexbox
- CSS Custom Properties (Variables)
- Canvas API
